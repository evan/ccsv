
Ccsv

A pure-C CSV parser.

== Installation

  gem install ccsv

== Usage

  require 'rubygems'
  require 'ccsv'

  Ccsv.foreach("data.csv") do |line|
    # Do something with the line array
  end

  # Or split on tabs, etc.
  Ccsv.foreach("data.tab","\t") do |line|
    # Do something with the line array
  end
